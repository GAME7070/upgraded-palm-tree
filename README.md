学生成绩管理系统
学号：25134060132 | 序号：29
项目简介
这是一个纯C语言实现的轻量控制台学生成绩管理工具，无需额外依赖即可在Windows/Linux环境下编译运行，支持学生信息录入、多维度查询、成绩统计排序、本地数据持久化全流程操作。
核心功能清单

学生信息管理：支持学号、姓名、3门科目成绩的录入，自动校验学号重复
多方式查询：支持按学号精准查询、按姓名模糊匹配搜索
数据统计：自动计算单科/总分平均分，统计及格率，支持按总分降序自动排序
持久化存储：所有录入数据自动写入本地student.dat文件，程序重启后数据不会丢失
异常防护：内置非法输入过滤，避免程序崩溃溢出问题

编译&运行指南
Windows环境（当前你的Legion R900P设备适配）

打开MinGW命令行工具，进入项目代码所在文件夹
执行编译命令生成可执行文件：gcc main.c -o studentManager.exe -Wall


双击运行生成的studentManager.exe即可操作，编译日志会自动输出所有告警信息。

Linux环境

终端进入项目目录
执行编译命令：gcc main.c -o studentManager -Wall


执行./studentManager直接启动程序。
